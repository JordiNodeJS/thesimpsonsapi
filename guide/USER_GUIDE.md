## Feature Guides

### Episode Tracker

#### Browsing Episodes

![Episodes Page](screenshots/04-episodes-page.png)

The **Episodes** page displays all episodes in a vertical list format:

- **Large episode posters** - Visual thumbnails for each episode
- **Episode title** - Full episode name
- **Season & Episode number** - E.g., "S1 E1"
- **Synopsis** - Complete episode description

**How to use:**
1. Click on any episode card to view details
2. Scroll through the list to browse all 768+ episodes
3. Episodes are organized sequentially by air date

#### Tracking Individual Episodes

![Episode Detail Page](screenshots/05-episode-detail.png)

When you click an episode, you can:

**Rate the Episode**
- Click 1-5 stars to rate the episode
- Your rating is saved to your personal profile

**Add Personal Notes**
- Use the "What did you think?" textarea to write your thoughts
- Save notes about favorite moments, characters, or quotes
- Click "Save Progress" to store your notes

**Share Trivia**
- Add "Did You Know?" facts about the episode
- Submit trivia using the text area
- Facts appear for all community members

**Navigation:**
- Use the **EPISODES** link in the nav bar to return to the full list
- Browse between episodes using browser back/forward

### Character Profiles

#### Browsing Characters

![Characters Page](screenshots/02-characters-page.png)

The **Characters** page features:

- **Recently Viewed** - Horizontal scrollable list of characters you've recently visited
- **Character Grid** - All 1,182+ characters in a responsive grid (2 columns mobile, 3 tablet, 4 desktop)

Each character card shows:
- Character image
- Character name
- Occupation/role (e.g., "Safety Inspector," "Student at Springfield Elementary")

**How to use:**
1. Scroll through the grid to browse characters
2. Click any character card to view their profile
3. Previously viewed characters appear in the "Recently Viewed" section

#### Character Detail Pages

![Character Detail Page](screenshots/03-character-detail.png)

**Character Profile Information:**
- Large character image
- Character name and occupation
- Bio section with character background

**Interactive Features:**

**2. Community Wall**

  - User avatar/initial
  - Username
  - Date posted
  - Comment text

**3. Share Trivia**
- Add "Did You Know?" facts about the character
- Type a trivia fact in the text area
- Click "Submit Fact" to share with the community
- Existing trivia displays with the submitter's name

**Example:**

## Tips & Tricks

### Getting the Most Out of Springfield Life

**1. Start with Your Favorite Episodes**
- Begin by tracking episodes you've already seen
- Rate them and add notes about why you love them
- Build your viewing history over time

**2. Follow Main Characters First**
- Start with the Simpson family
- Follow characters that appear in your favorite episodes
- Expand to supporting characters as you explore

**3. Use the Diary for Rewatches**
- Document your thoughts during rewatches
- Create entries for each viewing session
- Track how your opinions change over time

**4. Create Themed Collections**
- Organize quotes by character
- Create season-specific collections
- Build topical collections (e.g., "Best Comebacks," "Life Advice")

**5. Engage with the Community**
- Leave comments on character walls
- Share obscure trivia facts
- Read what other fans have contributed

**6. Keep Data Synced**
- Use the "Sync Data from API" button on the home page
- Ensures you have the latest episodes and characters
- Keeps the database up to date

## FAQ

### General Questions

**Q: Do I need to create an account?**  
A: Currently, the app uses a default "SimpsonsFan" user for testing. Full authentication is planned for future releases.

**Q: Is my data saved?**  
A: Yes! All ratings, notes, follows, and comments are stored in a PostgreSQL database (powered by Neon).

**Q: Can I access this on mobile?**  
A: Yes! The app is fully responsive and works on mobile browsers. A native mobile app is planned for the future.

**Q: Where does the episode/character data come from?**  
A: Data is sourced from The Simpsons API and stored locally for fast access.

### Feature-Specific Questions

**Q: How do I know which episodes I've already tracked?**  
A: Future updates will add visual indicators on the episodes list page. Currently, you can check individual episode pages to see if you've rated them.

**Q: Can I unfollow a character?**  
A: Yes! Click the "Following" button again to unfollow.

**Q: Can I edit or delete my comments?**  
A: This feature is planned for future updates.

**Q: How many collections can I create?**  
A: There's no limit! Create as many as you'd like.

**Q: Can I export my data?**  
A: Data export functionality is planned for future releases.

### Technical Questions

**Q: What technologies power this app?**  
A: Next.js 16, React 19, Tailwind CSS 4, Neon (PostgreSQL), shadcn/ui, Radix UI

**Q: Is this open source?**  
A: Yes! Licensed under MIT. Visit the [GitHub repository](https://github.com/JordiNodeJS/thesimpsonsapi).

**Q: Can I contribute?**  
A: Absolutely! Contributions are welcome. See the repository for guidelines.

**Q: Is there an API I can use?**  
A: The app uses The Simpsons API. Check the repository docs for API details.
