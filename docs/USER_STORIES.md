# User Stories - Springfield Life & Tracker

## üì∫ Episode Tracking
- **US-1.1:** As a user, I want to mark an episode as "Watched" so I can keep track of my progress.
- **US-1.2:** As a user, I want to rate an episode (1-5 stars) to express how much I liked it.
- **US-1.3:** As a user, I want to add personal notes/reviews to an episode for future reference.
- **US-1.4:** As a user, I want to view my progress by season to see how much of the show I've completed.

## ü§ù Springfield Social
- **US-2.1:** As a user, I want to "Follow" characters so I can show my support for them.
- **US-2.2:** As a user, I want to leave comments on character profiles to interact with other fans.
- **US-2.3:** As a user, I want to see an activity feed of characters I follow (simulated).

## üìú Quote Collector
- **US-3.1:** As a user, I want to create custom collections (e.g., "Homer's Wisdom") to organize quotes.
- **US-3.2:** As a user, I want to add quotes to my collections with their source.

## üí° Trivia Wiki
- **US-4.1:** As a user, I want to see "Did you know?" facts for Characters and Episodes.
- **US-4.2:** As a user, I want to submit my own trivia facts to contribute to the community.

## üìñ Springfield Diary (Simulator)
- **US-5.1:** As a user, I want to log "Daily Activities" in Springfield by selecting a Location and a Character.
- **US-5.2:** As a user, I want to write a journal entry describing my fictional interaction.
- **US-5.3:** As a user, I want to view a timeline of my past diary entries.

## üîç Discovery & Sync
- **US-6.1:** As a user, I want to browse a list of all characters and view their detailed profiles.
- **US-6.2:** As a user, I want to browse a list of all episodes and view their details.
- **US-6.3:** As a user, I want to sync the application with the external Simpsons API to ensure I have the latest data.
